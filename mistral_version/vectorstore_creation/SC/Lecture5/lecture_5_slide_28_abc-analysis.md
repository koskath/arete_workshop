# Slide 28 of Lecture 5 contains information about ABC Analysis,Pareto Principle or 80/20 Rule

Idea: ABC analysis separates out the most important items so that more attention can be focused on those items. It is based on the principle that out of the myriad of items an inventory manager needs to handle, there are only a few that account for most of the inventory expenses.

Checklist: Calculation of the total consumption value of each material per period (quantity multiplied by the cost price). Calculate percentage with respect to the total number of consumed goods. Calculate percentage with respect to the total consumption value of all goods. Rank material types in descending order with respect to the total consumption value. Cumulate these percentages with respect to the total number of consumed goods. Cumulate the percentage of the total consumption value of all goods. Classification of material types in A, B and C goods.
