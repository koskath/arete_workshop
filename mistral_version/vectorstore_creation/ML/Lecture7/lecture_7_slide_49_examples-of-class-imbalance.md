# Slide 49 of Lecture 7 contains a table with Examples of Class Imbalance.
# Here is what the table in slide 49 of lecture 7 contains:

### Columns:
- **SL**: Serial number  
- **Name**: Dataset name  
- **Data Types**: Type of data (e.g., multivariate)  
- **Default Task**: Typical ML task (classification/regression)  
- **Attribute Types**: Types of attributes (categorical, integer, real)  
- **#Instances**: Number of samples  
- **Class Distribution**: Samples per class  
- **#Attributes**: Number of features  
- **Imbalance Ratio**: Degree of class imbalance
