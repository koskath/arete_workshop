## Horizontal Process Flow
The diagram shows a sequential flow across four main stages:
- **Planning** → **Dispatching Scheduling** → **Control** → **Phys. Flow**

## Dimensional Analysis Across Process Stages

### Time Frame
- **Planning**: "long run", "anticipating"
- **Dispatching Scheduling**: "short run"
- **Control**: (Middle timeframe)
- **Phys. Flow**: (Execution phase)

### Hierarchical Position
- **Planning**: "high – top down"
- **Dispatching Scheduling**: (Transition level)
- **Control**: "middle"
- **Phys. Flow**: (Operational level)

### Critical Element
- **Planning**: "Handle Uncertainty"
- **Dispatching Scheduling**: "Select from Alternatives"
- **Control**: (Monitoring phase)
- **Phys. Flow**: (Physical execution)

### Solution By
- **Planning**: "Collecting Info", Setting premises, Building scenarios"
- **Dispatching Scheduling**: (Selection process)
- **Control**: "calculating Optima"
- **Phys. Flow**: (Implementation)

### Types of Info., Data Needed and Applied
- **Planning**: Historical data, patterns; Forecasts, parameters; Context Information
- **Dispatching Scheduling**: "Situative" data; Capacities, lead-times
- **Control**: (Performance data)
- **Phys. Flow**: (Execution data)
