# Slide 40 of Lecture 1 contains information about the Terminology.

Algorithm: is a set of procedures that creates a model when trained. E.g., linear regression. Model: is a fitted algorithm that has been trained. E.g. a linear regression model that has been trained to predict prices of 'X'. Parameters are the internal variables within a model that are adjusted automatically as you train a ML model. Hyperparameters are set by the user to control the algorithm and they define how it learns from the data. 40
